`# Bava 👂🇳🇱

**Live ondertiteling app — speciaal voor dove en slechthorende gebruikers, geoptimaliseerd voor Nederlands.**

## Gebruiken

Open op je iPhone: **https://icvanee.github.io/bava**

Voeg toe aan je thuisscherm: Safari → Deel-knop → "Zet op beginscherm"

## Functies

- 🎙️ **Real-time ondertiteling** — Spraak direct naar tekst
- 📺 **TV Modus** — Grote ondertitels voor TV kijken
- 📝 **Geschiedenis** — Sla gesprekken op, kopieer of deel ze
- 🌙 **3 Thema's** — Donker, Licht, Hoog Contrast
- 🔤 **Lettergrootte** — Van 16px tot 64px instelbaar
- 📳 **Trilsignalen** — Voel wanneer iemand praat
- 🔇 **Volume meter** — Zie het geluidsniveau live
- 📱 **PWA** — Installeerbaar op je thuisscherm
- 🇳🇱 **Nederlands** — Standaard `nl-NL` spraakherkenning

## Technologie

Gebouwd als Progressive Web App (PWA) met:
- Web Speech API (`nl-NL`)
- Service Worker voor offline UI
- LocalStorage voor transcripten
- Wake Lock API (scherm blijft aan)
- Web Share API

> **Let op**: Spraakherkenning op iOS Safari vereist een internetverbinding (audio wordt verwerkt door Apple's servers).
